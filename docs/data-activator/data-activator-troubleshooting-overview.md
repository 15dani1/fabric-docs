# Troubleshoot Data Activator Errors

If a problem occurs with any of your Data Activator events, objects, or triggers, then Data Activator will send you an email alert containing an error code. The articles in this section explain the meaning of the error codes that you can receive, and describe steps you can take to fix the problems.
